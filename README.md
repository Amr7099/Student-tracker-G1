# Student-tracker-G1 Report
## Description
This project is a simple calculator program that determines a student’s grade category based on their score.
It classifies the grade as Good, Very Good, or Excellent depending on the mark. (without manual calculation)

## Features
- Takes a student's score as input.
- Calculates and displays the grade category.
- Supports three grade levels:
  - Good: 40–59
  - Very Good: 60–74
  - Excellent: 75–100
    
## Technology Used
- Language: Python 
- Tool: GitHub for version control
  
## How It Works
1. The user enters their grade (0–100).
2. The program checks which range the grade belongs to.
3. It displays whether the grade is Good, Very Good, or Excellent.
   
## How to Run the Program
1. Clone the repository or download the program file.
2. Open it in your code editor or IDE.
3. Run the program and enter a grade to test.
